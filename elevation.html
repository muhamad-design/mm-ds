Foundations

Elevation

Depth comes from borders and tonal surfaces first. Three quiet shadow tiers cover cards, menus, and modals.

Borders first#

Hierarchy comes from borders and tonal surfaces before shadows, so the shadows that remain are quiet. The cards below use the live values - flip the theme to compare. Pair each tier with the matching radius from Shapes.

Tiers#

shadow-card
Raised cards and subtle lifts. Tooltips use this tier too.
L 0 2px 2px rgba(0, 0, 0, 0.04)D 0 1px 2px rgba(0, 0, 0, 0.16)
shadow-menu
Popovers, dropdown menus, and other transient surfaces.
L 0 1px 1px rgba(0, 0, 0, 0.02), 0 4px 8px -4px rgba(0, 0, 0, 0.04), 0 16px 24px -8px rgba(0, 0, 0, 0.06)D 0 1px 1px rgba(0, 0, 0, 0.02), 0 4px 8px -4px rgba(0, 0, 0, 0.04), 0 16px 24px -8px rgba(0, 0, 0, 0.06)
shadow-modal
Modals and dialogs - the deepest shadow in the system.
L 0 1px 1px rgba(0, 0, 0, 0.02), 0 8px 16px -4px rgba(0, 0, 0, 0.04), 0 24px 32px -8px rgba(0, 0, 0, 0.06)D 0 1px 1px rgba(0, 0, 0, 0.02), 0 8px 16px -4px rgba(0, 0, 0, 0.04), 0 24px 32px -8px rgba(0, 0, 0, 0.06)
