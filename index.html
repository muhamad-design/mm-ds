mm-ds

A Geist-based design system for mm products: minimal, high-contrast, light and dark. Token-for-token aligned with Vercel's public Geist reference, and readable by people and AI agents alike.

Foundations#

Colors

Ten-step scales where every step has a job: backgrounds, borders, fills, text.

Typography

Geist Sans and Geist Mono across heading, label, copy, and button tokens.

Materials

Borders first, then three quiet shadow tiers and the focus ring.

Components#

Interface building blocks rendered live from the same variables this site ships.

Avatar Badge Banner Button Checkbox Code Block Empty State Input Keyboard Input Note Pagination Progress Radio Select Skeleton Spinner Switch Table Textarea Tooltip

Consuming the system#

Web surfaces can link the generated custom properties directly; agents and tools read the markdown specs or the YAML token files. The full rules for layout, shapes, motion, voice, and the do's and don'ts live in the specs.

/* web */
<link rel="stylesheet" href="assets/tokens.css">
color: var(--c-gray-1000);
background: var(--c-background-100);
border-radius: var(--rounded-sm);
