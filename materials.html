Materials

Presets for radii, fills, strokes, and shadows.

Surface#

On the page.

material-baserounded-sm · gray-alpha-400
The resting material: a translucent border, no shadow.
material-smallrounded-sm · shadow-card
Raised cards and subtle lifts.
material-mediumrounded-sm · shadow-menu
Emphasized surfaces that stay in the page flow.
material-largerounded-md · shadow-modal
The strongest on-page emphasis.

Floating#

Above the page.

material-tooltiprounded-sm · shadow-card
Tooltips take the lightest floating material.
material-menurounded-md · shadow-menu
Popovers, dropdowns, and other transient surfaces.
material-modalrounded-md · shadow-modal
Modals and dialogs above a scrim.
material-fullscreenrounded-lg · shadow-modal
Sheets and fullscreen surfaces.

Focus ring#

Focus is a two-layer ring: a 2px gap in the surface color, then a 2px blue ring. It shows on every interactive element at :focus-visible.

Values

  • Light: box-shadow: 0 0 0 2px #ffffff, 0 0 0 4px #006bff (blue-700).
  • Dark: box-shadow: 0 0 0 2px #000000, 0 0 0 4px #47a8ff (blue-900).

Best Practices#

Choosing a material

  • Match the material to the layer: surface materials stay in the page flow, floating materials sit above it.
  • Depth comes from borders and tonal surfaces first; shadows stay quiet.
  • Tooltips always take the lightest floating material.
  • Pair modal and fullscreen materials with a scrim behind them.
